## E2E Flow Verification

### Flow 1: Authentication (10 steps)
**Status:** COMPLETE

1. Web app generates 6-digit code via `/auth/extension-code`
2. User copies code to extension popup
3. Extension validates format and sends AUTH_LOGIN
4. Background exchanges code for JWT via `/auth/extension-token`
5. Background fetches user info via `/auth/extension-me`
6. Token and user stored in chrome.storage.local
7. STATE_UPDATED broadcast triggers UI update
8. App.tsx renders ConnectedIndicator
9. Subsequent API calls use token via fetchWithAuth
10. 401 responses clear auth and prompt reconnection

### Flow 2: Data Selection (11 steps)
**Status:** COMPLETE

1. ProjectSelector sends GET_PROJECTS message
2. Background calls fetchProjects API
3. Projects rendered in dropdown
4. User selects project → PROJECT_SELECT message
5. Selection stored, triggers mapping check
6. BatchSelector sends GET_BATCHES message
7. Background calls fetchBatches API with projectId
8. Batches filtered (exclude completed) and rendered
9. User selects batch → BATCH_SELECT message
10. Selection stored with batchId, totalRows
11. Row indicator shows "Row 1 of N"

### Flow 3: Fill Cycle (20 steps)
**Status:** COMPLETE

1. Tab activation triggers checkMappingForTab
2. fetchMappingsByUrl called with project and URL
3. Valid mappings (stepCount > 0) filtered
4. Badge set with count on extension icon
5. MappingSelector shown in popup if hasMapping
6. User clicks Fill → FILL_START message
7. fetchMappingWithSteps gets full mapping
8. fetchRowByIndex gets current row data
9. FILL_EXECUTE sent to content script
10. content.ts calls executeSteps
11. Steps executed in order (fill/click/wait)
12. Result returned (success, stepResults)
13. Defensive undefined check prevents crash
14. updateRowStatus called with VALID or ERROR
15. If successTrigger configured, MONITOR_SUCCESS sent
16. SuccessMonitor watches for trigger condition
17. SUCCESS_DETECTED sent on trigger match
18. Auto-advance to next row
19. Or user manually clicks Next → ROW_NEXT
20. Cycle repeats for next row
