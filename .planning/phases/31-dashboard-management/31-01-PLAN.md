   - Enter key: call onSave with current value

   - Escape key: revert to original value, exit edit mode
6. Error handling:
   - Wrap onSave in try/catch
   - On error: stay in edit mode, show error message below input (use red text-destructive)
   - Store error in local state, clear on next save attempt
7. Loading state:
   - While onSave is pending, show Loader2 spinner next to input
   - Disable input during save

Use shadcn/ui Input component for the text input.

Component should be "use client".
