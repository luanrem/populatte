  <name>Task 2: Update RowIndicator component with identifier display</name>
  <files>
    apps/extension/entrypoints/popup/components/RowIndicator.tsx

  <action>
Rewrite the RowIndicator component per CONTEXT.md decisions:

**Props interface:**
```typescript
interface RowIndicatorProps {
  rowIndex: number;
  rowTotal: number;
  identifierPrimary: string | null;
  identifierSecondary: string | null;
  identifierFieldKey: string | null;  // For tooltip label
  secondaryFieldKey: string | null;   // For tooltip label
}
```

**Display layout (per CONTEXT.md):**
- Row number on top, identifiers stacked below
- Entire block has subtle background (e.g., `bg-gray-50 rounded-lg p-2`)
- Primary identifier: bolder/larger text (e.g., `font-medium text-gray-900`)
- Secondary identifier: lighter/smaller (e.g., `text-sm text-gray-600`)
- Row number: secondary (e.g., `text-xs text-gray-500`)

**Copy behavior:**
- Entire block is clickable (cursor-pointer)
- On click, copy primary identifier value to clipboard (fall back to doing nothing if only row number)
- Show inline feedback: brief "Copied!" text or CheckCheck icon for 1.5s
- Use `navigator.clipboard.writeText()` for copy

**Truncation:**
- Max width ~250px or text-ellipsis with overflow-hidden
- Tooltip on hover shows full value + field label
- Use `title` attribute for simple tooltip, or custom tooltip if needed

**Empty states (per CONTEXT.md):**
- No identifiers configured (both keys null): Show row number only, no identifier section
- Primary has value, secondary empty: Show primary only, hide secondary line
- Primary is empty but configured: Fall back to row number only

**Example structure:**
```tsx
<div
  className="bg-gray-50 rounded-lg p-2 text-center cursor-pointer hover:bg-gray-100 transition-colors"
  onClick={handleCopy}
  title={tooltipText}
>
  {/* Row number - always shown, secondary emphasis */}
  <div className="text-xs text-gray-500">
    Row {rowIndex + 1} of {rowTotal}
  </div>

  {/* Identifiers - only shown when configured and have values */}
  {identifierPrimary && (
    <div className="mt-1">
      <div
        className="font-medium text-gray-900 truncate max-w-[250px] mx-auto"
        title={`${identifierFieldKey}: ${identifierPrimary}`}
      >
        {copied ? 'Copied!' : identifierPrimary}
      </div>
      {identifierSecondary && (
        <div
          className="text-sm text-gray-600 truncate max-w-[250px] mx-auto"
          title={`${secondaryFieldKey}: ${identifierSecondary}`}
        >
          {identifierSecondary}
        </div>
      )}
    </div>
  )}
</div>
```

**State management:**
- `const [copied, setCopied] = useState(false)`
- Reset copied state after 1.5s timeout
- Cleanup timeout on unmount
